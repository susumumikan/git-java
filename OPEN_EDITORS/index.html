git push -u 付け忘れ対策

使用方法

pwd 現在のディレクトリを検索

git --version　現在のバージョンを検索

git init　gitを初期化

git add .　gitをadd（コミットするファイルを準備するところです。ワークツリーからインデックスに登録する）する

git commit -m "練習中”の”　ソースコードの更新作業がcommit”"この間はgithubコメントを残せる　

git push -u origin main 一回目のpushするときだけは-uを付ける

git push 次からはpushでアップできる

.gitignore　を作ってみた
